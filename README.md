## Gendiff

The CLI util for compares two files and shows a difference.

**Supported file extensions:**

- `.json`
- `.yml` | `.yaml`

**Formats output:**

- `stylish`
- `plain`
- `json`

## Install

```bash
git clone https://github.com/orishko/backend-project-lvl2.git
make install
```
